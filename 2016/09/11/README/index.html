以.开头的是交换文件

Linux下vi编辑文件时会产生一个交换文件，用于保存文件，当文件非正常退出时，可用此文件进行恢复，当文件正常关闭时，系统会自动删除 .swp时一个隐藏文件 使用ls -a查看

以只读方式打开([O]), 直接编辑((E)), 恢复((R)), 删除交换文件((D)), 退出((Q)),中止((A)):

可以使用R恢复文件 或者 vi -r 文件名 然后将.swp文件删除 rm .文件名.swp

正常退出vi编辑器是 shift ZZ

SSH key 是为了确认你是项目的管理者或拥有者，然后可以不使用密码使用git。
